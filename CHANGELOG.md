# 更新日志

> 本项目仍处于活跃维护中。

---

## [1.1.0] - 2026-04-27

### 优化
- **移除随机延迟功能**：删除分段发送时的随机延迟逻辑，消息分段现在连续发送，体验更即时
- **精简配置项**：移除 `random_delay_range` 配置项，简化用户配置界面
- **代码瘦身**：清理 `config_manager.py` 和 `main.py` 中冗余的 docstring 注释，提升代码可读性

### 保留
- **多平台打字状态**：`_set_typing` 方法继续支持 QQ(aiocqhttp)、Telegram、Discord 等平台的"正在输入"状态显示

---

## [1.0.0] - 2026-04-XX

### 新增
- 消息智能分段发送：根据配置的分段符号和长度阈值自动拆分长文本
- 字符串替换功能：支持自定义文本替换规则（如"小笨蛋=>小可爱"）
- Markdown 清除功能：自动清理加粗、代码块等 Markdown 格式标记
- 排除规则：支持关键词和正则表达式匹配，特定内容跳过处理
- 多平台打字状态：支持 QQ、Telegram、Discord 等平台的输入状态显示
- 短尾合并：最后一段过短时自动合并到前一段，避免碎片化
